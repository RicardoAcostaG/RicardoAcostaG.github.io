Introducción a JavaScript

Aprenda JavaScript, un lenguaje de programación de desarrollo web, para agregar interactividad a sus páginas web y conviértase en un desarrollador web experto.


Sobre este curso

Este curso es parte del Certificado Profesional de "Desarrollador Web Front-End" del W3C. Este curso de informática es una iniciación a la programación de JavaScript y ha sido diseñado para ayudar a los desarrolladores web a comprender los conceptos básicos del lenguaje. Este curso fue desarrollado en asociación entre el W3C y la Universidad Côte d'Azur. JavaScript le permite agregar funciones interactivas a sus sitios web, incluido contenido actualizado dinámicamente, multimedia controlada, imágenes animadas y mucho más. El objetivo principal de este curso es dominar las mejores prácticas de JavaScript mediante muchos ejemplos interactivos, algunos de los cuales se demuestran en videos de codificación en vivo. Utilizaremos JavaScript en el navegador web. Por qué vale la pena JavaScript:

HTML5, CSS y JavaScript son los "tres clásicos" para desarrolladores y diseñadores; Le permite agregar interactividad a sus sitios web; Puede usar JavaScript y HTML5 API para crear gráficos y animaciones personalizados, y para dominar multimedia usando reproductores de audio y video, música y efectos de sonido; Es poderoso, fácil de aprender y rápido de escribir; Tiene excelentes herramientas (editores, tiempos de ejecución, herramientas de pelusa, navegadores y bibliotecas de terceros), así como un excelente soporte en línea a través de muchas comunidades activas de código abierto.

Al final del curso, esperamos que pueda leer el código fuente de cualquier ejemplo de JavaScript que se encuentre en la Web, aprender de él, modificarlo e incluso, ¿por qué no? - Comience a contribuir a proyectos JavaScript de código abierto. Este curso introductorio te hará pensar como un desarrollador de JavaScript.
