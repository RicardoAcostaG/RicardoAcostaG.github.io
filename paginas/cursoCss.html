Conceptos básicos de CSS

Aprenda cómo llevar sus páginas web de cosas a atrevidas con el poder del estilo CSS.


Sobre este curso

Este curso es parte del Certificado Profesional de "Desarrollador Web Front-End" del W3C. ¿Acaba de comenzar su viaje hacia el desarrollo web? Este curso le presentará los conceptos básicos del diseño web y le enseñará cómo diseñar su sitio web para una apariencia profesional. El equipo de Microsoft y los expertos del World Wide Web Consortium (W3C) lo guiarán paso a paso en el uso de los últimos estándares web para estructurar su contenido y presentación como un profesional. En este curso, explorará numerosas funciones de CSS para ayudarlo a controlar la apariencia de su sitio, como los colores, las fuentes y el diseño. Al final del curso, debe comprender todos los elementos fundamentales de CSS y cómo usarlos de manera efectiva. Se recomienda que esté al menos un poco familiarizado con HTML antes de tomar este curso, pero le proporcionaremos el código HTML cuando sea necesario.
